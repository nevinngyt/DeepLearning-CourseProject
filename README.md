# DeepLearning-CourseProject

## Introduction & How to reproduce results

This github repo serves as the codebase for the Deep Learning project in the course 24788 in CMU. This project is about comparing baseline GNN against DimeNet. The main codebase used for model training is provided in "qm9_29April.ipynb" file. Note, all code is run on Google Collab. To reproduce the results without model training, please navigate to the reproduce folder and download everything into Collab. The "reproduce.ipynb" file could be run to reproduce the MAE values on the test dataset, training loss curves and the t-SNE figures. Please note to change the file paths for the csv files (For the train loss curves) and the model checkpoints based on your own file directory organisation. Pelase note that running the "reproduce.ipynb" file would take around 30 minutes.
